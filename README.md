# 🌌 Atmosfera — Weather App 2.0

A modern weather application built with JavaScript and Fetch API, featuring a redesigned glassmorphism interface, animated background and enhanced user experience.

## 🚀 Technologies
- HTML5  
- CSS3  
- JavaScript (ES6+)  
- Fetch API  
- Open-Meteo API  

## ✨ Features
- Search for any city in real-time  
- Display current weather conditions  
- 5-day weather forecast  
- Dynamic weather icons based on conditions  
- Animated background with floating orbs  
- Glassmorphism UI design  
- Responsive layout for all devices  
- Loading and error state handling  
- Empty state interface  

## 🎨 UI & Experience Improvements (v2.0)
- Complete UI redesign with glassmorphism effects  
- Smooth animations and transitions  
- Improved layout and spacing  
- Better visual hierarchy  
- Enhanced user feedback  

## 🖥️ How to Run
1. Clone or download this repository  
2. Open the `index.html` file in your browser
